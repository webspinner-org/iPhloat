# Inputs — the three offerings

A Webspinner site is born from three offerings the author provides. This directory holds them for iPhloat.

| Offering | Where it goes | What it is |
|---|---|---|
| **Imagery** | [`imagery/`](imagery/) | Photographs, illustrations, and brand assets that carry the visual voice. The build agent extracts colors, palettes, and rhythms from imagery to drive the site's CSS — the design honors what was given. |
| **Prose** | [`prose/`](prose/) | Documents the author writes with care — the substance of the brand, the tone, the things a patron must know. One file or many; markdown or plain text. |
| **Intent** | [`intent.txt`](intent.txt) | A single sentence that says, plainly, what this site is for. |

Drop files into the appropriate location. The Webspinner build agent reads from here when training the loom, gating the release, and rendering the site.

These are the same three offerings every Webspinner site receives. See [`../HOW-WEBSPINNER-WORKS.md`](../HOW-WEBSPINNER-WORKS.md).
