# iPhloat backlog

Public engineering issues for the iPhloat brand site live in this
repository's GitHub Issues. The product roadmap (what the
patron-facing site will offer next) is summarized below.

## Status

Pre-launch. The marketing surface, the brand-voice chat panel,
and the invited-tester sign-in gate are working. The cart and
checkout surfaces are being rehearsed before patrons see them.

## Roadmap (patron-facing)

| Surface | State |
|---|---|
| Brand site (marketing pages) | Live to invited testers |
| Brand-voice chat ("Ask iPhloat") | Live to invited testers |
| Product detail pages | In progress |
| Cart + checkout | Pre-launch |
| Customer support chat | Pre-launch |
| Order tracking | Pre-launch |
| Reviews + testimonials | Pre-launch |

For brand-voice and product-positioning intent, see
[`INTENT.md`](INTENT.md). For the engineering of the fleet of
agents that build and operate the site, see [`AGENTS.md`](AGENTS.md).
